إعداد التخزين على GitHub

خطوات الإعداد

للحصول على تطبيق يعمل مع تخزين دائم للبيانات، اتبع هذه الخطوات:

الخطوة 1: إنشاء Personal Access Token
  1. اذهب إلى GitHub Settings
  2. انقر على "Generate new token"
  3. اختر "Generate new token (classic)"
  4. أدخل اسم: "Ice Cream Shop Storage"
  5. اختر الأذونات: repo
  6. انقر على "Generate token"
  7. انسخ الـ token واحفظه
الخطوة 2: إنشاء مستودع GitHub
  1. اذهب إلى GitHub New Repository
  2. أدخل اسم المستودع: golden-ice-cream-shop
  3. اختر "Public" أو "Private"
  4. انقر على "Create repository"
  5. أنشئ مجلد data في المستودع
  6. ارفع ملفات التطبيق
الخطوة 3: إدخال بيانات الإعداد
تنبيه مهم

إعدادات GitHub ستُحفظ في المتصفح فقط ولن تظهر لأي شخص آخر. بياناتك آمنة ومحمية.
